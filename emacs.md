#Emacs

##Cursor Motion

###Characters
- ```C-f``` move forward
- ```C-b``` move backward
- ```C-d``` delete forward
- ```DEL``` delete backward

###Words
- ```M-f``` move forward
- ```M-b``` move backward
- ```M-d``` delete forward
- ```M-DEL``` delete backward

###Lines
- ```C-n``` move forward
- ```C-p``` move backward
- ```C-k``` delete forward
- ```C-SPC C-a C-w``` delete backward

###Sentences
- ```M-a``` move forward
- ```M-e``` move backward
- ```M-k``` delete forward
- ```C-x DEL``` delete backward
